# Customer Segmentation Analysis (Power BI)

## Project Overview
This project analyzes customer spending behavior using SQL, Python, and Power BI. The data was extracted and queried using SQL, cleaned and explored using Python, and an interactive dashboard was built in Power BI to analyze customer segments based on age, gender, and income.

## Tools Used
- Power BI
- Python (Pandas, data cleaning & analysis)
- SQL (data extraction and querying)
- CSV / Excel

## Key Features
- Data extraction and filtering using SQL
- Data cleaning and preprocessing using Python
- Customer segmentation based on spending behavior
- Gender-wise, age-wise, and income-wise analysis
- Interactive dashboards with slicers

## Dataset
Shopping Mall Customer Segmentation Dataset

## Outcome
The dashboard provides clear insights into customer behavior and helps identify key customer segments for data-driven business decision-making.
